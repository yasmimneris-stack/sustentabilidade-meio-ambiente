# 🌍 Sustentabilidade e Meio Ambiente

Projeto desenvolvido para o **Concurso Agrinho 2026**, com o tema **"Sustentabilidade e meio ambiente"**.

#agrinho2026

## 🎯 Objetivo do Projeto

O Agrinho é um programa educativo que aproxima estudantes de temas importantes para a sociedade e para o futuro do planeta. Este projeto tem como objetivo mostrar, de forma leve, visual e interativa, que pequenas atitudes do dia a dia fazem diferença na preservação do meio ambiente.

O site apresenta:
- O conceito de sustentabilidade explicado de forma simples;
- Atitudes práticas para o dia a dia (economia de energia, de água, consumo consciente e reciclagem);
- Uma calculadora simples de pegada de carbono a partir dos quilômetros rodados de carro por semana;
- Um jogo interativo de separação de lixo, com pontuação em tempo real.

## 💡 Justificativa

Falar sobre sustentabilidade é essencial para formar cidadãos mais conscientes do impacto de suas escolhas no meio ambiente. Ao apresentar o tema de forma prática e interativa, o projeto busca engajar o visitante a repensar pequenos hábitos que, somados, geram um grande impacto positivo para o planeta.

## 🛠️ Tecnologias Utilizadas

- **HTML5** — estruturação semântica do conteúdo (formulários, botões, seções, links);
- **CSS3** — estilização, layout com Flexbox e Grid, variáveis de cor e Media Queries para responsividade;
- **JavaScript (ES6)** — interatividade: saudação personalizada, modo escuro, controle de tamanho de fonte, calculadora de pegada de carbono e jogo de separação de lixo com pontuação.

## ▶️ Como Usar

1. Baixe ou clone este repositório.
2. Abra o arquivo `index.html` em qualquer navegador (não precisa de internet nem instalação).
3. Digite seu nome na tela inicial para receber uma saudação personalizada.
4. Use os botões no topo da página para ativar o modo escuro ou ajustar o tamanho da fonte.
5. Role a página para conhecer as atitudes sustentáveis, testar a calculadora de pegada de carbono e jogar o jogo de separação de lixo.
